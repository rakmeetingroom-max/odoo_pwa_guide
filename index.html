Odoo Mobile: Fix Frequent Sign-Out Issues Using PWA

IT Advisory · RAK Real Estate · For All Staff
Prepared by: IT / Network Services Date: April 2026  |  Applies to: iPhone users (iOS + Chrome)

The Problem

Users accessing Odoo via the App Store app often experience frequent sign-outs and session drops. The recommended fix is to switch to the PWA (Progressive Web App) version — accessed through Chrome and added to your Home Screen. The PWA always runs the latest version and maintains a more stable session.

App Store App vs. PWA — What's the Difference?

❌ App Store App (Avoid)

May run outdated version · Prone to session timeouts · Requires manual updates

✅ PWA — Progressive Web App (Recommended)

Always up-to-date · More stable login sessions · Looks and feels like a native app

How to Install the PWA on iPhone (3 Steps)

1

Open Chrome & Sign In

Open Google Chrome (not Safari) and go to:

https://rakrealestate.odoo.com/

Sign in with your Odoo credentials.

2

Tap the Share Icon

Tap the Share icon (box with arrow) at the top-right of the Chrome address bar.

3

Add to Home Screen

Scroll down in the share menu and tap "Add to Home Screen". Confirm to install.

Done — use this icon going forward.

Result: A new RRE ERP icon appears on your Home Screen. Use this instead of the App Store version. The PWA icon (left) is your new shortcut — the App Store version (right) can be removed.

Key Takeaways

Action Impact
✔ Install PWA via Chrome Replaces the App Store app with a more stable, always-updated version
✔ Use Chrome (not Safari) Required for PWA installation on iPhone — Safari does not support this flow
✔ Sign in once via PWA Session is retained longer, reducing repeated login prompts
✔ Remove old App Store app (optional) Avoids confusion between two Odoo icons on your device
